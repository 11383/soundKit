Have fun with 'metronome' object in console ;)

Basic usage

  • new Metronome({ beats: 4, tempo: 60, preset })
  • metronome.setTempo(120)
  • metronome.setMetrum(4)
  • metronome.play()
  • metronome.stop()

Events

  • metronome.on('beat', () => { console.log('I will show on every beat') })
  • metronome.on('beat_1', () => { console.log('I will show only on first beat') })
  • metronome.on('sound', (sound) => { console.log('I will show on every sound', sound) })
  • metronome.on('loaded', (metronome) => { console.log('Metronome loaded') })
